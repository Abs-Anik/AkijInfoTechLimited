Access Point (AP)

An Access Point (AP) provides wireless connectivity, enabling devices to connect to a network.

Aruba AP-515 WI-FI 6 Indoor Access Points

  • Main Features
  • Model: Aruba AP-515 WI-FI 6 Indoor Access Points
  • Connectivity: Indoor
  • WAN Port: 10/100/1000BASE-T

Specification

Specification Details
Brand Hewlett-Packard Enterprise (HPE)
Country of Origin U.S.A
Model Aruba AP-515 Wi-Fi 6 Indoor Access Points
AP Type Indoor
Wi-Fi Radio Specifications
  • Dual radio, 5GHz 802.11ax 4x4 MIMO and 2.4 GHz 802.11ax 2x2 MIMO
  • Support for up to 512 associated client devices per radio, and up to 16 BSSIDs per radio
  • 5GHz radio: Up to 4.8Gbps wireless data rate with 4x4 MU-MIMO
  • 2.4GHz radio: Up to 574Mbps wireless data rate with 2x2 SU-MIMO
Supported Radio Technologies
  • 802.11b: Direct-Sequence Spread-Spectrum (DSSS)
  • Low-Density Parity Check (LDPC) for high-efficiency error correction
  • 802.11mc Fine Timing Measurement (FTM) for precision distance ranging
  • 802.11ax Target Wait Time (TWT) to support low-power client devices
  • Transmit Beam-Forming (TxBF) for increased signal reliability and range
Wi-Fi Antennas
  • Four integrated dual-band downtilt omnidirectional antennas for 4x4 MIMO (2x2 MIMO for 2.4GHz)
  • Peak antenna gain of 4.2dBi in 2.4GHz and 7.5dBi in 5GHz
  • Optimized for horizontal ceiling mounted orientation
Certifications
  • Wi-Fi CERTIFIED a, b, g, n, ac, ax
  • WPA, WPA2 and WPA3—Enterprise with CNSA option, Personal (SAE), Enhanced Open (OWE)
  • WMM, WMM-PS, Wi-Fi Agile Multiband
  • Bluetooth SIG
  • Wi-Fi CERTIFIED Location™ and Passpoint (release 2)
Other Interfaces and Features
  • 10/100/1000BASE-T Ethernet network interface (RJ-45)
  • Auto-sensing link speed and MDI/MDX
  • 802.3az Energy Efficient Ethernet (EEE)
  • USB 2.0 host interface (Type A connector)
Power Sources
  • The AP supports Power over Ethernet (PoE)
  • Maximum power consumption in idle mode: 12.6W (PoE) or 9.7W (DC)
  • Maximum power consumption in deep-sleep mode: 5.9W (PoE) or 1.5W (DC)
Wi-Fi Features
  • Transmit Beamforming
  • Passpoint Wi-Fi (Release 2) (Hotspot 2.0)
  • Dynamic Frequency Selection (DFS)
  • Maximum Ratio Combining (MRC)
  • Cyclic Delay/Shift Diversity (CDD/CSD)
  • Space-Time Block Coding
  • Low-Density Parity Check (LDPC)
Warranty Limited Lifetime Warranty
